# 🚀 LFS-init

## 📖 Visão Geral
Inicializador de novos projetos do Lab Fabrica de Software com estrutura pré-definida

## ⚙️  Uso

### Usar sem instalar localmente

```bash
npm init @labfs/init
```

###  Instalar localmente
```bash
npm install -g @labfs/init
```

**Utilizar como cli**

```bash
lfs
```
